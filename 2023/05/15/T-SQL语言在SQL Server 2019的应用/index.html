T-SQL语言在SQL Server 2019的应用

1. SQL语言概述

SQL（Structured Query Language）语言是利用一些简单句子构成的基本语法，来存取数据库的内容。

2. T-SQL的特点

1）一体化特点
2）T-SQL具有交互式和嵌入式两种使用方式
3）高度非过程化、采用集合的操作方式
4）语言简洁，易学易用

SQL功能 命令动词
数据定义（数据模式定义、删除、修改） CREATE、DROP、ALTER
数据操纵（数据查询和维护） SELECT、INSERT、UPDATE、DELETE
数据控制（数据存取控制授权、收权） GRANT、REVOKE
5）支持三级数据模式结构

常用完整性约束
  • 唯一性约束：UNIQUE
  • 非空值约束：NOT NULL
  • DEFAULT约束，缺省约束，
    格式为：DEFAULT<默认值>
  • 主码约束：PRIMARY KEY
    保证主码的唯一性与非空性，可直接写在主码后，或按语法单独列出。
  • 参照完整性约束
  • CHECK约束
SELECT语句介绍

数据查询的基本语法格式如下：

select子句：指定要显示的属性列。
from子句：指定查询的数据源（基本表、视图或连接表）。
where子句：指定查询条件。
group by子句：对查询结果按指定列的值分组，该属性列值相等的元组为一个组。通常会在每组中作用SQL统计函数。
having短语：筛选出只有满足指定条件的组。
order by子句：对查询结果表按指定列值的升序或降序排序。
